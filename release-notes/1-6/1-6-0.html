Release Notes

A new version of New Tab Tweaker has been installed.

What's new in 1.6.0
  • Any wallpaper URL that is prefixed with "[list]" will be treated as a dynamic provider of image URLs in plaintext. This can be used together with HFS as a workaround to allow for local files to be used as wallpapers .
  • Allows import of wallpaper URLs from Imgur galleries (in addition to albums).
  • Adds a new wallpaper scaling option: 'Fill'.
